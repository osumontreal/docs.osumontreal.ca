Credits

We appreciate all efforts that go into making tools, packages and libraries, and documentation freely available to all. So here’s a non-exhaustive list dedicated to the ones we have used so far to build and support the Open Science UMontreal community.

Thanks to them for making Open Science and Open Source Software better!

Documentation

  • Matrix at TU Dresden - The team behind Matrix at TU Dresden has provided a wonderful documentation available under a CC BY-SA 4.0 that we adapted to our needs. Refer to our Matrix Documentation section for an overview.

Tools

  • Hugo - A fast and flexible static site generator written in Go, which we use for all our websites.
  • GitHub - Continuous deployment and hosting all of our repositories, including this documentation.
  • Matrix - An open network for secure, decentralized communication freely available.

Packages and libraries

  • hugo-theme-learn - A multilingual-ready theme fully designed for documentation that was the starting point of our actual documentation theme (this website). It includes libraries such as
    • lunr - Lunr enables you to provide a great search experience without the need for external, server-side, search services…
    • horsey - Progressive and customizable autocomplete component;
    • clipboard.js - copy text to clipboard;
    • highlight.js - A Javascript syntax highlighter;
    • modernizr - A JavaScript toolkit that allows web developers to use new CSS3 and HTML5 features while maintaining a fine level of control over browsers that don’t support;
    • mermaid - Generation of diagram and flowchart from text in a similar manner as markdown;
    • jQuery - The Write Less, Do More, JavaScript Library;
    • font awesome - the iconic font and CSS framework.
